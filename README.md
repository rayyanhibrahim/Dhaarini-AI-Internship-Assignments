# Machine Learning Mini Projects

This repository contains the machine learning mini-projects completed during the **first week** of my **AIML Internship** at **Dhaarini AI-Tech Research Academy**. The projects provide hands-on experience with supervised and unsupervised machine learning algorithms using real-world datasets.

## 📚 Projects Included

### MP01 – Linear Regression
- Dataset: Bike Sharing (hour.csv)
- Objective: Predict continuous values using Linear Regression.

### MP02 – Logistic Regression
- Dataset: Give Me Some Credit
- Objective: Perform binary classification using Logistic Regression.

### MP03 – Decision Trees
- Dataset: Emotion Classification
- Objective: Build a Decision Tree classifier for emotion prediction.

### MP04 – Ensemble Learning
- Dataset: Employee Attrition
- Objective: Predict employee attrition using ensemble learning techniques.

### MP05 – K-Means Clustering
- Dataset: Online Retail
- Objective: Segment customers using the K-Means clustering algorithm.

### MP06 – DBSCAN Clustering
- Dataset: Credit Card Customers
- Objective: Identify customer groups and outliers using DBSCAN.

### MP07 – Principal Component Analysis (PCA)
- Dataset: Marketing Campaign
- Objective: Reduce data dimensionality using Principal Component Analysis.
